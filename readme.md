# Lathello

Lathello is a web application that can play Reversi against AI based on the AlphaZero algorithm.

# Features

- Play against AlphaZero
- You can also write posts

# Project Stack

## Client

- React
- React Router
- TypeScript
- SWR
- Tailwind CSS
- Emotion

## Server (Laravel)

- PHP
- Laravel
- Google OAuth 2.0

## Server (for AlphaZero)

- Python
- TensorFlow
- Keras
- NumPy
- Flask
